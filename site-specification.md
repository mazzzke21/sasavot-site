## ГЛОБАЛЬНЫЕ ВИЗУАЛЬНЫЕ ЭФФЕКТЫ (CSS / Canvas overlay)

   - Статическая текстура — горизонтальные линии разной толщины и насыщенности

   - Периодически (раз в 20–40 секунд) проскакивает более яркая полоса — имитация «сбоя отслеживания» ленты

   - RGB-split (каналы красного и синего смещаются относительно зелёного)

3. **CRT-эффект (Cathode Ray Tube):**
   - Виньетка по краям экрана (радиальное затемнение)
   - Едва заметное мерцание (flicker)
   - Scanlines (горизонтальные линии развёртки, opacity: 0.02)

4. **«Грязь» на плёнке (Film grain / dust):**
   - Наложение текстуры старой киноплёнки с точками, царапинами, ворсинками
   - opacity: 0.04

5. **Курсор:**
   - Обычное состояние: стандартная стрелка, но с небольшим красным свечением (drop-shadow: #cc0000)
   - При наведении на имена жертв или фото: курсор меняется на значок «битой кассеты» (иконка VHS с трещиной), либо на «стоп-кадр» ✋

### Рекомендации по audio-контенту

1. **VHS_HUM.wav** — 60 Hz hum, записанный с реального VHS-проигрывателя, зацикленный, -25dB
2. **TAPE_HISS.wav** — белый шум, фильтрованный (high-pass 8kHz), -30dB
3. **Glitch SFX** — короткий (50-150ms) резкий цифровой артефакт, -15dB
4. **Voice clips** — обработанные через вокодер, с эффектом «старого радио» (band-pass 300Hz-3kHz)
5. **Whisper reversed** — шёпот, записанный неизвестным диктором, прокрученный задом наперёд (исходная фраза не поддаётся восстановлению — так указано в файле)

### Рекомендации по реализации видео-плеера

- Не использовать стандартный HTML5-плеер — слишком чистый
- Наложение Canvas поверх `<video>`, на котором отрисовываются VHS-эффекты:
  - Случайные искажения цвета (HSV-shift)
  - Полосы отслеживания (random horizontal bars)
  - Дрожание кадра (random 1-3px offset каждые 50-200ms)
  - Белый шум при «обрыве» (noise overlay на последних секундах)
  - Макро-блокинг (large pixel blocks) при ухудшении качества

### Анимации и переходы

- **Переход между разделами:** fade через чёрный (500ms) + звук перемотки кассеты
- **Появление контента:** плавное появление с нарастанием белого шума, который спадает до 0 за 1 секунду
- **Спойлер-блоки:** раскрытие с эффектом «разворачивания старой бумаги» (CSS transition + box-shadow)
- **Карта:** появление точек-маркеров с пульсацией, последовательно (сначала dasha, затем остальные с интервалом 500ms)
- **Финальный раздел:** при первом посещении — полный сброс CSS на 2 секунды (белый экран), затем восстановление
